chrono

Think of chrono in C++ as your program’s built-in clock + stopwatch + calendar


Quick Info Box:


Header: <chrono>

Type: Utilities

Best for: Measuring execution time, delays






Concept


Start → (Running Code) → End
"""""""""""""""↓
""""" (Duration = End - Start)


In other words, chrono library lets you measure the time in the program.

Here is an example for further comprehension-



Example:

#include <iostream>
#include <chrono>

int main() {
,,,,,,,auto start = std::chrono::high_resolution_clock::now();

,,,,,,,for(int i = 0; i < 1000000; i++){
,,,,,,,,,,,,,,// some work
,,,,,,,}

,,,,,,,auto end = std::chrono::high_resolution_clock::now();

,,,,,,,auto duration = std::chrono::duration_cast
,,,,,,,<std::chrono::millisecond>(end - start);

,,,,,,,std::cout << duration.count() << " ms\n";
,,,,,,,return 0;
}

Output

2 ms

// this is the time it takes to execute this program

Real Use Case


Chrono is used when you need to delay some functions, to measure program's speed etc.
In other words, when you need to manipulate timing in your projects, Chrono is very efficient.
Some use case:


  1. Game Development (FPS & Cooldown):

    Control time inside a game -

    Shoot → wait 2 seconds → shoot again


    ◼ Add cooldowns(no spamming actions)

    ◼ Maintain constant FPS(smooth gameplay)

    Without chrono → game feels laggy or too fast.




  2. Measuring Code Speed (Performance)

    Find out which code is faster -

    Code A ⚙️ → 5 ms

    Code B ⚙️ → 2 ms ✅ faster


    ◼ Compare algorithms

    ◼ Optimize slow programs

    Used by developers to make apps faster.




  3. Backend / Server Timing:

    Do not wait forever for a response -

    Request → wait (max 3s) → timeout ❌


    ◼ Set timeout for APIs

    ◼ Measure response time

    Prevents apps from freezing.




  4. Multithreading Control

    Pause or schedule tasks -

    Task 1 ✅

    (wait 1 second)

    Task 2 ✅


    ◼ Delay execution

    ◼ Run tasks at intervals

    Used in background workers, schedulers




  5. Timers & Alarms

    Do something after (X) time -

    Start ⏱️ → 10 seconds → Ring 🔔


    ◼ Countdown timers

    ◼ Reminder systems

    Like stopwatch or alarm apps




  6. Animation & UI Timing

    Make things move smoothly -

    Position → update every 16ms → smooth motion


    ◼ Control animation speed

    ◼ Sync visuals with time

    Used in games, apps, UI effects




Important Note

Different clocks exist in C++.
For most cases, high_resolution_clock is used for precise timing.
