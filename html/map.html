map

An Associative container that stores data as key-value pairs in sorted order


Quick Info Box:


Header: <map>

Type: Container(STL)

Best for:Storing key-value pairs






How it works


Key,,,,,,,,,,,,,,,,,,Value


Apple ,,,, → ,,,,,,50

Banana , → ,,,,,,30

Mango ,, → ,,,,,,20



A map stores data as key-value pairs.

The key identifies an item.
The value stores information associated with that key.

Each key must be unique.

Here is an example for further comprehension-



Example:

#include <map>;
#include <iostream>;

int main(){
,,,,,,,std::map<std::string, int> fruits;

,,,,,,,fruits["Apple"] = 50;
,,,,,,,fruits["Banana"] = 30;
,,,,,,,fruits["Mango"] = 20;

,,,,,,,std::cout << fruits["Apple"];
,,,,,,,return 0;
}

Output

50

// prints only the value of the key "Apple" of the map called fruits






Common Operations


  • insert() → Add a new key-value pair


    Apple → 50

    """ ↓

    insert(Orange → 25)

    """ ↓

    Apple → 50
    Orange → 25


    Example:""fruits.insert({"Orange", 25});




  • access ([]) → Get the value using its key


    Apple → 50

    fruits["Apple"] → Retrieve Value → 50


    Example:""fruits["Apple"];




  • find() → Searches the data by its key


    Apple
    Banana
    Mango
    '''''↓
    find(Banana)
    '''''↓
    [Searches the map]
    '''''↓
    Banana[✓ Found]


    Example:""fruits.find("Banana");




  • erase() → Delete a key-value pair


    Apple → 50
    Banana → 30
    Mango → 20
    '''''↓
    erase(Banana)
    '''''↓
    [Removes the Banana key-value pair]
    '''''↓
    Apple → 50
    Mango → 20


    Example:""fruits.erase("Banana");


  • Loop Through All Elements


    ┌─────────────┐
    │ Apple 50.................. │
    ├─────────────┤
    │ Banana 30...............│
    ├─────────────┤
    │ Mango 50.................│
    └─────────────┘
    .................↑
    Iterator visits each pair


    Code:

    for(auto pair : fruits)
    {
    std::cout << pair.first << " : " << pair.second << "\n";
    }

    Output:

    Apple : 50
    Banana : 30
    Orange : 25





Real Use Case


Map is very useful in real-world projects.
It is used when your program needs a container to store data oderly or to access data.
Some use case:


  1. Student Database:

    1001 → Alice

    1002 → Bob

    1003 → Charlie



    ◻︎ Student ID = Key

    ◻︎ Student Name = Value




  2. Word Counter:

    "hello" → 5

    "world" → 3


    ◻︎ Count how many times each word appears




  3. Game Scoreboard:

    "Player1" → 1200

    "Player2" → 850


    ◻︎ Store player scores




  4. Product Inventory:

    "Laptop" → 15

    "Mouse" → 40

    "Keyboard" → 25


    ◻︎ Track how many items are available.




  5. Phone Book:

    "John" → "017xxxxxxxx"

    "Sarah" → "018xxxxxxxx"


    ◻︎ Find a phone number using a name.




Important Note


Keys in std::map are always sorted.
